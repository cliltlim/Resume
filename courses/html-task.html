Задание: практика по HTML

  • Создайте 3 ссылки с условиями

    • Каждая ссылка должна вести на любую поисковую систему
    • Каждая ссылка должна быть оформлена в виде прямоугольника размером 200 на 50 пикселей
    • Каждая ссылка должна иметь фон синий цвет и белый текст
    • Между ссылками должен быть вертикальный отступ в 10 пикселей
    • Все ссылки должны иметь сплошную красную границу толщиной 2 пикселя

  • Создайте форму для выбора хобби:

    • Создайте форму, которая будет содержать список хобби: кино, музыка, спорт
    • Используйте элементы <label> и <input> для создания формы
    • Для каждого хобби добавьте флажок checkbox, который позволит пользователю
      выбрать несколько значений

  • Задание: Изменение текста с помощью DOM

    • Создайте веб-страницу с текстом, который отображается внутри элемента <div>
    • Напишите кнопку, при нажатии на которую, текст в элементе <div> изменится на "Текст был изменен!"
    • Используйте JavaScript и DOM для изменения содержимого страницы без перезагрузки
